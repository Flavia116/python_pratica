# Visualização de Dados - Aulas

Este repositório reúne os exercícios e práticas que estou desenvolvendo durante meu estudo de Visualização de Dados com Python, usando as bibliotecas:

- Seaborn
- Matplotlib
- Pandas
- NumPy

## Estrutura

| Aula | Conteúdo abordado |
|------|--------------------|
| Aula 01 | Gráficos com o dataset `tips` do Seaborn |
| Aula 02 | Regressões lineares, KDEs, boxplot, violinplot e geração de dados aleatórios |

## Objetivo

Aprender e praticar conceitos fundamentais de visualização de dados para análise exploratória, comunicação de resultados e preparação para projetos futuros.
